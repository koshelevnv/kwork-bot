# Kwork Parser Bot

Телеграм-бот мониторит новые заказы (вкладка «Хочу заказать») на kwork.ru и присылает уведомления. Каждый пользователь настраивает свои категории, ключевые слова и фильтры прямо в боте.

---

## Быстрый старт

### Linux VPS (рекомендуется)

> Недорогой и стабильный VPS от 119 рублей в месяц — [play2go.cloud](https://play2go.cloud/?ref_id=TAiMBIAReXI)

**1. Создай бота и получи токен**

Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен вида `1234567890:ABC...`

Свой Telegram ID можно узнать через [@myidbot](https://t.me/myidbot) → `/getid`

**2. Запусти установку одной командой**

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/koshelevnv/kwork-bot/main/manage.sh)
```

Скрипт спросит только токен бота, затем автоматически:
- Установит Python, git, screen
- Клонирует репозиторий
- Создаст виртуальное окружение и установит зависимости
- Создаст `.env` с токеном
- Запустит бота в фоновой screen-сессии
- Откроет меню управления

После установки для управления ботом:

```bash
~/kwork-bot/manage.sh
```

Поддерживаемые дистрибутивы: Ubuntu/Debian, CentOS/RHEL, Fedora, Alpine, Arch.

#### Управление через manage.sh

| Пункт | Описание |
|---|---|
| 1. Запустить | Запуск в screen-сессии (автоустановка при первом запуске) |
| 2. Статус и логи | Состояние бота, последние строки лога |
| 3. Остановить | Остановить бота |
| 4. Обновить | Обновить код из GitHub и перезапустить |
| 5. Выйти | Выйти из скрипта (бот продолжает работать) |

Вручную присоединиться к сессии: `screen -r kwork_bot`

### Windows

1. Установи Python 3.11 с [python.org](https://www.python.org/downloads/) (галочка **Add Python to PATH**)
2. Создай бота у [@BotFather](https://t.me/BotFather) и получи токен
3. Скопируй `.env.example` → `.env`, заполни токен
4. Запусти `start.bat` (при первом запуске скачает зависимости ~1 мин)

## Настройка .env

Скопируй `.env.example` в `.env` и заполни:

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdef...
```

| Параметр | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) (обязательно) |
| `ADMIN_IDS` | Telegram user_id через запятую — принудительные администраторы (необязательно) |

> Первый пользователь, написавший `/start`, автоматически становится администратором.
> Свой Telegram ID: [@myidbot](https://t.me/myidbot) → `/getid`

## Управление через бота

После запуска нажми **🚀 Запустить бота** или напиши `/start`.

### Основные кнопки

| Кнопка | Описание |
|---|---|
| 📋 Мои категории | Просмотр и удаление отслеживаемых категорий |
| 🎛 Фильтры | Добавить категорию, фильтр по цене, ключевые слова |
| 📊 Статус | Все текущие настройки |
| ⚙️ Настройки | Частота, период, дни недели, часовой пояс, цена |
| 🔄 Перезапустить бота | Обновить приветствие и кнопки |

---

### 🎛 Фильтры

**Добавить категорию** — выбери раздел → выбери категории из списка.
- Клик без галочки — добавляет (✅)
- Клик на ✅ — удаляет
- Можно ввести ID вручную (берётся из URL: `kwork.ru/projects?c=41` → ID = `41`)

**Ключевые слова** — фильтрация заказов по тексту.
- Список пуст → проходят все заказы из категорий
- Список не пуст → приходят только заказы, содержащие хотя бы одно слово

**Фильтр по цене** — ограничение по бюджету заказа. Заказы без указанной цены проходят всегда.

---

### ⚙️ Настройки

**⏱ Частота уведомлений** — пресеты: 30 сек / 1 мин / 90 сек / 2 мин / 3 мин / 5 мин / 10 мин.

**🕐 Период уведомлений** — в какие часы присылать уведомления. Поддерживается переход через полночь (например, 22:00 – 06:00). По умолчанию: круглосуточно.

**📅 Дни недели** — в какие дни присылать уведомления. Каждый день включается/выключается отдельно. По умолчанию: Пн–Пт.

**🌍 Часовой пояс** — UTC-offset из списка. По умолчанию UTC+3 (Москва).

**💰 Цена** — диапазон бюджета заказов. По умолчанию: без ограничений.

## Администратор

В разделе **⚙️ Настройки** у администратора дополнительный блок:

| Кнопка | Описание |
|---|---|
| 👥 Пользователей: N | Общее количество зарегистрированных пользователей |
| 🌐 Парсер: N сек | Глобальный интервал проверки kwork.ru (для всех пользователей) |
| 👥 Регистрация: открыта/закрыта | Запрет регистрации новых пользователей |
| 🔄 Обновить категории | Загрузить актуальный список категорий с kwork.ru |

> Регистрация по умолчанию закрыта. Открыть её может только администратор.

При регистрации нового пользователя администратор получает уведомление с именем, username и ID.
